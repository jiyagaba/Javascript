Variable Rules

  • Cannot be a reserved keyword (if, let, var)
  • Variable name should be meaningful
  • Cannot start with a number
  • Cannot contain space or underscore
  • Camel case example: (ageOfStudent), (loveBabba)

Multiple Variable Declaration

We can use let a, b; too (multiple variable declaration)

Types of Data

Primitive data type

  • Number (integer, float)
  • String (Sequence of characters)
  • Boolean (true, false)
  • Undefined (VALUE NOT DEFINED)
    • Example: 
      let a;
console.log(a); // undefined
  • NULL (VALUE IS EMPTY)

Dynamic Typing

We can add any type of data inside variables.

  • In CSS and Java: 
    int a = 5;
a = "Babbar"; // error
  • In JavaScript: 
    let a = 5;
a = "Babbar"; // No error

Reference Types

  • Objects (KEY: VALUE PAIR)
    • Objects have properties and behaviors
    • Example: 
      let Person = {
    firstname: 'Jiya',
    age: 24
};

      Here Person is an object. We can access the object using:

      • Dot Notation: Person.age
      • Bracket Notation: Person["age"]
  • Arrays
    • A data structure that contains a similar list of items.
    • Example: 
      let names = ['love', 'rahul', 'sangram'];
names[3] = "Sham";
  • Functions

Printing

We can do printing by console.log() function.

Var and let Keyword

Both keywords are used to create variables. Inside a variable, we can store any type of data.

  • Var and let are two keywords through which we can create variables.
  • Difference between var and let is just scope:
    • Let has local scope, whereas var has global scope.
    • Redeclaration is not possible with let, but it is possible with var.

Constant Variables

If we try to assign a new value to a constant variable, an error (Assignment to constant variable) will occur.
